Review of calculus through real analysis

Calculus is used everywhere in science and engineering. However, once going beyond the basics of single variable differentiation and integration, there just seems to be many variants of calculus that are used in practice coming out of nowhere. To name a few:

  • Vector and matrix calculus
  • Directional, partial, full derivative
  • Jacobian and Hessian matrix
  • Improper integral
  • Line and surface integral
  • Divergence and curl
  • …

In this article, I will try to give a consistent and unified view of different calculus concepts, which is known as real analysis. Calculus doesn’t stop at real analysis. As most of mathematical fields, different aspects of real analysis can be generalized and evolve into more advanced and complicated theory. Topics stemming from real analysis including complex analysis, differential geometry, general topology are out of my knowledge right now. However, I hope to cover the basics of real analysis so that the standard usage of calculus in classic mechanics and electromagnetism can be explained intuitively.

Limit

Calculus concerns limit behaviour. For example, in differentiation we want to know how does the function behave if we keep zooming into a particular point. In integration we want to know what does the sum approaches to if we keep adding small areas. Hence, we need to define what limit means.

When we say lim⁡x→tf(x)=l\lim_{x \to t} f(x) = llimx→t​f(x)=l, what we mean is that when xxx gets closer and closer to ttt, f(x)f(x)f(x) gets closer and closer to lll. Note the equal sign here actually means a much more involved predicate. More formally, we say:

∀ϵ>0,∃δ>0,d(x,t)<δ  ⟹  d′(f(x),l)<ϵ\forall \epsilon > 0, \exists \delta > 0, d(x, t) < \delta \implies d'(f(x), l) < \epsilon∀ϵ>0,∃δ>0,d(x,t)<δ⟹d′(f(x),l)<ϵ

It reads as, no matter how close (ϵ\epsilonϵ) we want to get close to lll, we can always find a range (δ\deltaδ), such that xxx in the range will make f(x)f(x)f(x) that close to lll. ddd and d′d'd′ are deliberately left unspecified so that there’s freedom to choose what they are, i.e. we only need some abstract measure of distance (known as metrics) for the idea of limit to be sensible.

Differentiation

Differentiation is all about local linear approximation of a function. The idea is, no matter how curvy the function looks like globally, when we zoom into a specific point enough, the curve will look flat. We want to know about this flat (linear) approximation locally, known as derivative.

For a function f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm, fff is differentiable at ttt if the following condition hold.

∃T:Rn→Rm,f(x)=f(t)+T(x−t)+R(x−t)  ⟹  lim⁡x→tR(x−t)∣x−t∣=0\exists T: \mathbb{R}^n \to \mathbb{R}^m, f(x) = f(t) + T(x - t) + R(x - t) \implies \lim_{x \to t} \frac{R(x - t)}{|x - t|} = 0∃T:Rn→Rm,f(x)=f(t)+T(x−t)+R(x−t)⟹x→tlim​∣x−t∣R(x−t)​=0

Here TTT is the derivative. The sentence basically means finding a linear TTT so that the difference R(x−t)R(x - t)R(x−t) between the actual value f(x)f(x)f(x) and the linearly approximated value f(t)+T(x−t)f(t) + T(x - t)f(t)+T(x−t) is going to somewhat approaching zero. That is, the linear approximation around ttt is good enough if we get close enough to ttt. We denote the derivative as DfDfDf, or dfdx\frac{df}{dx}dxdf​, or f′f'f′, or f(1)f^{(1)}f(1), or other notations.

We can analyse the signature of differentiation. The derivative itself is a linear map. For function f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm, the derivative at each point must be a function Rn→Rm\mathbb{R}^n \to \mathbb{R}^mRn→Rm. Therefore, if the entire function is differentiable everywhere, then the derivative of the entire function is of type Rn→(Rn→Rm)\mathbb{R}^n \to (\mathbb{R}^n \to \mathbb{R}^m)Rn→(Rn→Rm). Usually in single variable calculus we say the derivative is a number (usually interpreted as the slope of the graph). This is actually a simplification. A derivative of value 333 should be ×3\times 3×3 if we present the full form.

Variants of differentiation

In this section, I want explain how different forms of differentiation can be explained only through differentiation defined in the previous section plus some flexible view of functions.

For a function f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm, there’re nnn independent variables. For every y∈Rn−1y \in \mathbb{R}^{n - 1}y∈Rn−1, we can define a function g:R→Rmg: \mathbb{R} \to \mathbb{R}^mg:R→Rm such that g(x)=f(x,y)g(x) = f(x, y)g(x)=f(x,y). That is, we keep certain dimensions of input variables as constants. Similarly, we can also define g:Rn→R=πi∘fg: \mathbb{R}^n \to \mathbb{R} = \pi_i \circ fg:Rn→R=πi​∘f where πi\pi_iπi​ is the projection functions that extract coordinates from Rm\mathbb{R}^mRm.

The derivative of the resulting functions are called partial derivative, denoted as ∂fi∂xj\frac{\partial f_i}{\partial x_j}∂xj​∂fi​​. Partial derivatives are nothing but the normal derivatives restricted to certain dependent or independent variables. The concept of derivative is still the same as before. It’s still a linear map approximation, just now on a ‘partial’ function. With this understanding, we can actually calculate the derivative of any number of dependent variables against any number of independent variables.

In the similar spirit, we can also define directional derivative through functions. For example, given a function f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm, we can define another function g:R→Rmg: \mathbb{R} \to \mathbb{R}^mg:R→Rm given a point t∈Rnt \in \mathbb{R}^nt∈Rn and a direction v⃗∈Rn\vec{v} \in \mathbb{R}^{n}v∈Rn, so that g(x)=f(t+xv)g(x) = f(t + xv)g(x)=f(t+xv). Then the derivative is defined normally on ggg.

Frequently we need to do differentiation of ddt(a⃗⋅b⃗)\frac{d}{dt}(\vec{a} \cdot \vec{b})dtd​(a⋅b) or ddt(a⃗×b⃗)\frac{d}{dt}(\vec{a} \times \vec{b})dtd​(a×b), however dot and cross products are nothing but functions whose dependent variables are of type R\mathbb{R}R and R3\mathbb{R}^3R3, just the same as multiplication or addition are also functions. They are all functions, and there’s only one way to define differentiation on function. The rules used to calculate their derivatives shouldn’t confuse their nature of local linear approximation.

Two well-known matrix concept in differentiation are Jacobian and Hessian matrix, and they turned out to be nothing but derivative linear maps represented as matrix. For a function f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm, its Jacobian is just its derivative as m×nm \times nm×n matrix. Hessian matrix can only be defined for scalar functions f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R. It’s first derivative (also known as gradient) is of type f′:Rn→(Rn→R)f': \mathbb{R}^n \to (\mathbb{R}^n \to \mathbb{R})f′:Rn→(Rn→R), since Rn→R\mathbb{R}^n \to \mathbb{R}Rn→R is of dimension nnn, we can then find the derivative of f′f'f′ which is a n×nn \times nn×n matrix (the Jocobian of its first order derivative).

Matrix calculus are nothing but running differentiation on functions whose independent or dependent variables can be matrix. Since norms (known as operator norm) and metrics can also be defined on matrices, there’s nothing conceptually different from matrix differentiation and ordinary differentiation, except for the calculation rules get more complicated. Note that even with matrix differentiating against another matrix, the result can still be represented as a matrix, since derivative is always a linear map and linear maps can be represented as matrices. Similarly, higher order derivatives can just be seen as doing matrix calculus repetitively.

Integration

The most common interpretation of integration is to find the area under the graph of a function. However, this geometric interpretation might not apply if the concept of area doesn’t exist in a problem. Instead, a more algebraic interpretation of integration is summing up function values across certain domain. I prefer to use this language because domains and function values are general notions, and one can freely interpret the integration result (known as the integral) as area or curve length or any other quantity of interest, given integral suits the definition of the quantity.

Riemann integral is the most basic integration operation. Given a function f:Rn→Rf: \mathbb{R}^n \to \mathbb{R}f:Rn→R and its domain I=[a1,b1]×[a2,b2]×⋯×[an,bn]I = [a_1, b_1] \times [a_2, b_2] \times \dots \times [a_n, b_n]I=[a1​,b1​]×[a2​,b2​]×⋯×[an​,bn​]. To integrate over this domain, we partition this cubical domain into small grids of volume gig_igi​, and for each grid pick a random function value f(ti)f(t_i)f(ti​) for each grid, then the Riemann integral (if exists) is defined as:

∫If(x)=lim⁡max⁡gi→0∑if(ti)gi\int_{I} f(x) = \lim_{\max g_i \to 0} \sum_i f(t_i)g_i∫I​f(x)=maxgi​→0lim​i∑​f(ti​)gi​

That is, we partition the domain into sections. For each section, calculate how much value for that section then sum them up. Since we use a single point in each section as the function value for the entire section, the result will be inaccurate. However, it will become more and more accurate if we make the sections smaller and smaller, and the actual integral is the limiting value.

Variants of integration

An integral over higher dimensional domain Rn\mathbb{R}^nRn turns out can be evaluated from iterated application of integration over single dimension (justified by Fubini’s theorem). This is where usually the double or triple integration symbol comes from. For example, we reduce ∫Sf\int_S f∫S​f to ∬f(x,y)dxdy\iint f(x, y)dxdy∬f(x,y)dxdy, or reduce ∫Vf\int_V f∫V​f to ∭f(x,y,z)dxdydz\iiint f(x, y, z)dxdydz∭f(x,y,z)dxdydz. However, I think it’s important to realise that, conceptually the integration is still performed over the higher dimensional domain. The fact that we can evaluate it as iterated single dimensional integral is only of calculation benefit.

Another type of integral is improper integral, which calculates the integral over an unbounded domain. These integrals should be understood as the limit of a normal bounded integral. For example, ∫0∞f(x)dx\int_0^\infty f(x) dx∫0∞​f(x)dx should be understood as lim⁡b→∞∫0bf(x)dx\lim_{b \to \infty} \int_0^b f(x) dxlimb→∞​∫0b​f(x)dx. The underlying integration operation is exactly the same.

A natural question to raise about Riemann integral is, how do integrate over non-cubical domain? There’re different tricks for this problem, however, the general perspective I want to adopt is to use change of variable formula. That is, convert an integral of non-cubical domain ∫Sf\int_S f∫S​f, to an integral of a cubical domain with a transformation function ∫Cf∘ϕ\int_C f \circ \phi∫C​f∘ϕ, where ϕ\phiϕ transforms domain CCC to domain SSS.

Generalization to differential form

There are certain types of integrals that seem to be able to work with vector-valued functions. Two famous ones are line integral and surface integral. These integrals seem to be able to integrate dot product into the integration process. For example ∫CF⃗⋅dr⃗\int_C \vec{F} \cdot d\vec{r}∫C​F⋅dr or ∫SF⃗⋅dS⃗\int_S \vec{F} \cdot d\vec{S}∫S​F⋅dS.

One way to interpret these integrals is through parametrization and change of variables that converts their integration domains to the familiar cubical domains. However, a more sophisticated and generalized view is using differential forms. A thorough explanation on differential forms would need to involve concepts in differential geometry, so I only want to discuss the intuitions here.

Integration consists of breaking domain into small sections, find values of the sections, and sum them up. However, how to obtain the value of a section is actually up to us to define. So far we have been using elementary ‘section size ×\times× function value’ as the value, but this can be generalized to any function. This function will take in an infinitesimal section and return a value for the section. Differential form is just this function. That is, we integrate infinitesimal differentials together with the corresponding differential forms to combine and get the final integral result. In the case of line integral, the differential form would be f(dr⃗)=F⃗⋅dr⃗f(d\vec{r}) = \vec{F} \cdot d\vec{r}f(dr)=F⋅dr.

Line and surface integrals are used to define divergence and curl, which are just the limit of those integrals. Divergence is defined using the limit of surface integral while taking infinitesimal small surface around a point. Curl is defined using the limit of line integral while taking infinitesimal line around a point.
